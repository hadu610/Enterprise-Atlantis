Trust & Security

Built for the security review.

Enterprise buyers do not buy AI platforms; they buy auditable systems. Trust posture is the floor, not the ceiling. Every commitment below is architectural — not a roadmap promise.

🛡

SOC 2 Type II

Type II from day one. ISO 27001 by Month 18. HIPAA for healthcare verticals. Audit reports available under NDA.

🔑

Zero-trust agent identity

Every agent has its own scoped OAuth keys — never a sub-process of a human. Every action authenticated, scoped, time-bounded, and audited.

🏢

Multi-tenant isolation

Your data, wiki, audit log, billing — partitioned per tenant from the first line of code. Optional dedicated cell at higher tiers.

🔐

Customer-managed keys

Bring your own keys (BYOK) from Growth tier up. Rotate, revoke, or split-key at any time. Your encryption boundary, your control.

☁️

Bring-your-own-cloud

Optional. Run the agent runtime inside your AWS, GCP, or Azure tenant under your IAM. Our control plane, your data plane.

⏪

One-click rollback

Every state-changing action stores a snapshot. Delete and financial actions always require a human — hard rule, never configurable.

The new way to use enterprise software

One UI everyone already knows: the chat window.

Most enterprise SaaS forces every user to master a UI. Atlantis flips it. The default way to use the platform is to chat with an AI that has already mastered it — the Atlantis Manager. The Control Center UI is always available, but never required. Read the Manager spec →

Primary surface · chat

Atlantis Manager.

An orchestrator that sits above the seven department agents. Every employee — from day 1 — chats with the same Manager. Onboarding, daily work, investigation, configuration. The Manager produces a Resolution Plan before every action so you always see why.

  • Floats on every page (look bottom-right)
  • Knows the page you're on; routes you to the right surface
  • Never executes without a Plan + your approval
  • Sonnet 4.6 · prompt-cached · context-aware
Spectator surface · live

Live Activity Stream.

Most platforms show you what already happened (reports) or what needs doing (queues). Almost none show you the live execution itself. The Live Activity Stream is always on in every Autonomy Mode — you can sit back and watch your company run, like watching your civilization run in a real-time strategy game.
