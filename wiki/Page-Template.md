## Authoring checklist

Before submitting a new or updated page for approval:

- [ ] Filename uses `Title-Case-With-Hyphens`
- [ ] Metadata block is present and complete
- [ ] All field values match the [Wiki Conventions](Wiki-Conventions) enumeration
- [ ] `Status: Draft` (let the Owner promote to `Approved`)
- [ ] `Last reviewed` date is today (ISO format `YYYY-MM-DD`)
- [ ] Summary section is 1–3 sentences, no more
- [ ] Imperative voice for rules and playbooks; declarative for references
- [ ] Cross-references section is populated with at least one link
- [ ] No TODO markers in the body — use `Status: WIP` instead
- [ ] No customer-identifying information
